# 제품 요구 사항 문서 (PRD) - 성림교회 웹사이트

## 1. 프로젝트 개요

본 프로젝트는 성림교회 웹사이트를 구축 및 보완하는 것을 목표로 합니다. 기존에 구축된 기본적인 웹사이트 구조를 기반으로, 교회의 주요 활동과 정보를 효과적으로 전달하고 사용자 친화적인 인터페이스를 제공하는 기능을 추가하여 완성도를 높이고자 합니다.

## 2. 목표

*   교회 소개, 주일 말씀, 교회 학교, 교회 소식 등 핵심 정보를 효율적으로 제공.
*   관리자가 쉽게 콘텐츠를 업데이트할 수 있는 시스템 구축 (YouTube, WordPress 연동).
*   사용자가 필요한 정보를 쉽게 찾고 접근할 수 있는 직관적인 UI/UX 제공.

## 3. 핵심 기능 목록

#### 1. 환영 및 교회 소개 (Welcome / About)
*   **인사말:** 담임목사 사진(오른쪽)과 3문단 정도의 텍스트가 배치된 페이지.
*   **교회 비전:** 교회의 3가지 큰 비전을 중심으로 각각을 설명하는 페이지.
*   **교회 연혁:** 50주년 역사를 보여주는 타임라인 형태의 UI.
*   **섬기는 사람들:** 추후 위원회 조직도를 이미지로 간단히 추가할 수 있는 기본 틀.

#### 2. 주일 말씀 (Sermons)
*   **구현 방식:** YouTube Data API v3를 활용. 백엔드 서버가 API 키를 이용해 지정된 재생목록 ID로부터 영상 목록(제목, 썸네일, 영상 ID 등)을 주기적으로 가져와 프론트엔드에 전달.
*   **주일 설교 / 찬양대 / 특별 찬양:** 각각 별도의 재생목록을 사용하며, 동일한 메커니즘으로 구현. 관리자는 YouTube에서 재생목록에 영상을 추가하기만 하면 웹사이트에 자동 업데이트됨.

#### 3. 교회학교 (Education)
*   유아부, 아동부, 중고등부, 청년부 각 부서별 간단한 소개와 소식을 담을 수 있는 정적 페이지.

#### 4. 교회 소식 (News)
*   **연동 방식:** Headless WordPress의 REST API를 활용. 프론트엔드에서 데이터를 표시할 수 있는 UI 컴포넌트 틀을 구현. (API 연동 자체는 추후 직접 진행 예정)
*   **교회 소식:** 교회 행사 사진 등을 올리는 갤러리 형태의 게시판.
*   **주보:**
    *   **레이아웃:** 좌/우 분할 화면. 오른쪽은 역대 주보 제목 목록, 왼쪽은 선택된 주보 이미지를 표시.
    *   **관리 방식:** 관리자가 WordPress에 주보 이미지를 업로드하여 게시글을 생성하면, 웹사이트에서 목록을 불러와 표시.
*   **자유 게시판:** 현재는 읽기 전용 기능만 구현.
